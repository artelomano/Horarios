# Logo de Clínica Dental Jerez

## Instrucciones

Para agregar el logo de la clínica:

1. Descarga el logo desde https://www.clinicadentaljerez.net/
2. Guarda el archivo como `logo-clinica-dental-jerez.png` en esta carpeta (`public/images/`)
3. El logo se mostrará automáticamente en:
   - Página de login
   - Header del admin
   - Página pública

## Especificaciones recomendadas

- **Formato**: PNG con fondo transparente
- **Tamaño**: Máximo 250px de ancho
- **Nombre**: `logo-clinica-dental-jerez.png`

Si el logo no está disponible, la aplicación funcionará normalmente sin mostrarlo.
